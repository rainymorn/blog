QTimer will increase interval when App becoming background on OS X
2019-08-13| notes|

This is because OSX’s AppNap function,

One App will be sleeped by AppNap When it becoming background for a while.

this class will disable AppNap

how to use this class in Qt:

  1. modify .pro file

    • add this line

      OBJECTIVE_SOURCES += appnap.mm

    • del the appnap.mm from .pro file’s SOURCES Setting

      this setting was added automatically by ide when you create appnap.h file.

  2. add one variable of AppNapDisabler type in the main function

    AppNapDisabler appNap;

  3. Done

thanks this tutorial:

https://github.com/msorvig/qt-and-app-nap
